
### Instruction
- Clean:
<pre>./mvnw clean</pre>

- Compile:
<pre>./mvnw package</pre>

- Run gui mode:
<pre>java -jar target/swingy.jar gui</pre>

- Run console mode:
<pre>java -jar target/swingy.jar console</pre>
